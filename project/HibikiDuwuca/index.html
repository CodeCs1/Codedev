The first (?) Linux From Scratch Operating System that inspired by a Vietnamese Vtuber called

Hibiki Duca.

How To Install

There are 2 ways to do that:
#1: Download the ISO, boot up, login as 'root' (pw: root)
And then follow the instruction:
Mount main drive -> format with cfdisk or fdisk -> chroot -> install base component with
'duca -i base*'
#2: Download the duca's package manager follow the instruction:
[Same as #1] but instead of chroot, run this:
'duca -i base* -root=[mount root]'
